# الهوية البصرية ودليل تصميم واجهات منظومة كفيل السحابية 🌟

يركز هذا الدليل على مواصفات التصميم الفاخر والمظهر العصري المعتمد في **منظومة كفيل السحابية**، والذي يجمع بين البساطة الراقية للمظهر الأبيض النقي وفخامة اللون الذهبي المعدني اللامع، بالإضافة إلى المظهر الداكن الملكي الفخم.

---

## 🎨 1. لوحة الألوان الرئيسية (Visual Color Palette)

تعتمد المنظومة نظام ألوان متناسق مستوحى من المعادن الثمينة ليعطي شعوراً بالأمان والاحترافية:

| اللون | القيمة (HEX) | الاستخدام الأساسي | الأثر النفسي |
| :--- | :--- | :--- | :--- |
| **الذهبي الملكي** | `#bf953f` | التدرج اللامع للرأسيات والخطوط الفاصلة | الفخامة والتميز |
| **الذهبي الفاتح والساطع** | `#fcf6ba` / `#fbf5b7` | الإضاءات والانعكاسات على الحدود والأزرار | الرقي والوضوح |
| **النحاسي العميق** | `#aa771c` / `#b45309` | حدود الحقول والتنبيهات المكتوبة لزيادة التباين | الأمان والدقة |
| **الأبيض الناصع** | `#ffffff` | الخلفية العامة والبطاقات والنظام | النقاء والهدوء |
| **الرمادي الحريري** | `#f8fafc` / `#f1f5f9` | تباين بطاقات الخلفيات البسيطة والحقول | النظافة والترتيب |
| **الأسود الفحمي الملكي** | `#0f172a` / `#1e293b` | الوضع الداكن الفخم لإبراز العناصر المذهبة | الهيبة والقوة الأمنية |

## ✨ 2. عناصر التصميم الفاخرة (Premium UI Elements)

### 👑 المربع الذهبي العلوي المتناسق (Compact Gold Header)
تم دمج وتصغير المربع الذهبي ليكون شريطاً علوياً نحيفاً وثابتاً في أقصى الصفحة مع المحافظة على التدرج المعدني اللامع:
* **تأثير الخلفية:** `linear-gradient(135deg, #bf953f 0%, #fcf6ba 25%, #b38728 50%, #fbf5b7 75%, #aa771c 100%)`
* **الظل:** `box-shadow: 0 3px 15px rgba(170, 119, 28, 0.25)`

### 💧 الشعار الهافت المائي (Faded Watermark)
تم وضع شعار كفيل بشكل هافت جداً وجميل كعلامة مائية خلفية مكررة لا تشتت الانتباه وتضفي طابعاً براندياً راقياً:
* **الشفافية الدقيقة:** `0.07` في الوضع الفاتح، و `0.13` في الوضع الداكن.
* **الموقع:** تم ضبط محاذاة الخلفية وتعديل مؤشرات التراكم لضمان ظهور الشعار المائي بخلفية لوحة تحكم مدير النظام بشكل كامل وانسيابي دون تغطية أي محتوى.

### 🌟 الحدود الذهبية الفاتحة لكافة عناصر المنظومة (Global Golden Borders)
الابتكار الأبرز والأكثر فخامة في واجهة مستخدم المنظومة هو اعتماد الحدود الذهبية الفاتحة الراقية لكافة البطاقات، والحقول، والجداول، والأزرار، والفواصل، والأيقونات في شتى أنحاء النظام بشكل موحد:
```css
border: 1px solid #bf953f; /* لون ذهبي معدني فاتح للمحيط */
border-radius: var(--radius-lg); /* حواف دائرية انسيابية ممتازة */
box-shadow: var(--shadow-sm), 0 10px 30px rgba(212, 175, 55, 0.05); /* توهج ظلي ذهبي ناعم */
```

### 💫 تأثيرات الحركة التفاعلية الدقيقة (Micro-Animations)
عند تحريك مؤشر الماوس فوق الأزرار أو عناصر القائمة، تتفاعل الأيقونات بمرونة مذهلة:
* **الحركة التفاعلية:** تدور الأيقونة بمقدار 3 درجات وتكبر بنسبة 8% مع توهج ذهبي مشع:
```css
border-color: #fef08a;
background: rgba(251, 245, 183, 0.22);
transform: scale(1.08) rotate(3deg);
box-shadow: 0 2px 8px rgba(170, 119, 28, 0.35);
```

### 🌙 مفاتيح الهلال والشمس (Solar/Lunar Theme Toggles)
تم دمج أزرار تبديل المظهر التفاعلية (الهلال والشمس) في شريط التحكم العلوي الأيسر بالقرب من اسم المستخدم وفي بطاقات الدخول والاشتراك، مما يتيح التبديل اللحظي المتكامل لكامل ألوان وخلفيات المنظومة بين الثيم المظلم (خلفية فحمي داكنة) والمضيء (خلفية بيضاء ناصعة) مع الاحتفاظ بنقاء الحدود والأيقونات المذهبة وحفظ الاختيارات تلقائياً داخل الـ Local Storage.

### 👑 طاقم الإدارة العليا وسجل الرقابة الأمني (System Owners & Audit Panel)
لوحة فرعية فاخرة تعلوها أيقونات الدروع المذهبة والتي تضم:
* **طاقم المُلّاك**: قائمة منسابة لأصحاب الصلاحية العليا المعززين بأيقونات التاج الملكي الذهبي 👑 مع إمكانية التحكم الفوري في الصلاحيات.
* **سجل الرقابة الأمنية**: جدول زمني متفاعل يسجل لحظياً الإجراءات الإدارية الهامة (إنشاء المكاتب، تغيير الصلاحيات، تعديل باقات الاشتراكات) ببطاقات لونية ذكية.

### 👤 شريط الترحيب والتحكم الأيسر (Left Profile Welcome Block)
كتلة تحكم مدمجة في أعلى اليسار بإطار ذهبي فاتح `#d4af37` وخلفية داكنة راقية تعرض:
* جملة الترحيب الشخصية: `"مرحباً، مالك النظام"`.
* زر الانتقال السريع للإعدادات والتهيئة.
* زر تسجيل الخروج بلون أحمر تنبيهي دافئ وفاخر.

## 🏛️ 3. هيكلة القوائم المنسدلة الذكية (Smart Hover Dropdowns)

تم دمج القوائم المتعددة لتوفير مساحة العمل وسرعة التصفح، حيث تفتح تلقائياً بمجرد تمرير مؤشر الماوس بدون الحاجة للنقر:

1. **الاشتراكات والمستخدمين:**
   * إدارة المكاتب والاشتراكات.
   * المستخدمين النشطين (Active Users Hub).
   * طاقم الإدارة العليا والرقابة الأمنية (System Owners).
   * رموز الانضمام (أكواد التسجيل).
2. **الشؤون المالية:**
   * تعديل وبناء باقات الاشتراك (SaaS plans).
   * إدارة الموزعين والمسوقين ونسب عمولاتهم.
3. **النظام والاتصال:**
   * بوابة بث الإعلانات والتعميمات للمكاتب.
   * مركز استلام وحل تذاكر الدعم الفني للمكاتب.
   * النسخ الاحتياطي وحماية السيرفر (System logs & backups).
4. **تهيئة النظام:**
   * تخصيص الهوية والشعار (White-label settings).
   * بوابات الدفع الإلكتروني والـ SMS.
   * جهات العمل والمصارف وفروعها المعتمدة.

## 🔒 4. الضوابط الأمنية وقواعد البيانات

* يتم تخزين كافة البيانات في قاعدة بيانات سحابية متكاملة وآمنة مع تتبع العمليات الحساسة وتدوينها تلقائياً في سجل نشاط أمن النظام الخاص بالمدير العام.
* تم إدراج نظام تجميد الحسابات (Account Freezing) في واجهة المستخدم لإيقاف الحسابات المعرضة لمحاولات الدخول الخاطئ المتكرر بضغطة زر واحدة.
* يتم الحفاظ على كفاءة وأداء المنظومة عبر قياس زمن الاستجابة الفوري للشبكة للمحافظة على أسرع تجربة مستخدم ممكنة.
